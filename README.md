# Metrics Engine API
A backend service that accepts numeric datasets and returns descriptive
statistical metrics.

## Goals (v1)
- Accepts list of numeric values
- Compute:
  - Mean
  - Median
  - Variance
  - Standard Deviation
- Return structured JSON response

## Tech Stack
- FastAPI
- Python
- (NumPy optional in v1)

## Future Enhancements
- Named dataset storage
- Correlation analysis
- Regression
- Authenication
- Dockerization
